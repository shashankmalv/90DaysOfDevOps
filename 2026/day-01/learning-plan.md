# 90-day personal DevOps learning plan

## Current Level

#### I am Shashank,
A working professional with 3.10 years of experience in SRE and Observability tools, and a keen interest in transitioning to DevOps and Cloud Engineering.

## Why I am Starting to Learn DevOps & Cloud
I am starting to learn DevOps & Cloud to enhance my skill set, stay relevant in the evolving tech landscape, and open up new career opportunities in cloud-native technologies and infrastructure management.
## What is DevOps...
I understand DevOps as a culture where development and operations work together to build, test, deploy, and maintain applications smoothly. Cloud computing helps us use servers and services over the internet without managing hardware, which makes systems faster and scalable.

## Why..
I am learning DevOps and Cloud because I want to become job ready and confident in modern technologies. I want to move from theory to practical learning and build projects that I can show during interviews.

## How..
In the 90 days, my goal is to be confident with Linux basics, Git, and networking fundamentals. I also want to build project CI/CD pipelines, create Docker images, and deploy a simple application on Kubernetes using basic cloud services.

During this journey, I want to focus on building core DevOps skills. These are troubleshooting, finding solution, tools, Cloud (AWS and Azure), CI/CD pipeline creation, and Docker and Kubernetes or other related tools fundamentals. These skills will form the foundation of my DevOps career.

## Where I Want to Reach
In the next 90 days, I aim to:
1. Deploy a production-grade application on Kubernetes.

3. Develop strong Linux troubleshooting skills.
4. Create Infrastructure on the cloud through the IOC Tool.
